Constructor -
  • It is a member function of a class with same name as that of a class.
  • It is used to initialize data members of class.
  • It is always declared under public section.
  • It does not have return type and hence cannot return any value, not even void.
  • It is invoked automatically when object of class is created and initialize all data members of a class.
  • It cannot be declared as static, const and virtual.
Syntax:
class_name ()
{ Constructor Definition }

Example:

class example
{
   int x,y;
   public:
   example()
   {
      x=10;
	  y=20;
   }
   void display()
   {
      cout‹‹x‹‹y;
   }
};
int main()
{
  example O;
  O.display();
  return 0;
}
Output: 10  20

There 3 types of constructors, namely:
1)Default constructor - A constructor with no parameter is default constructor. If constructor is not defined in program then compiler supplies default constructor.
2)Parameterized constructor - A constructor that receives parameters is parameterized constructor.
3)Copy constructor - Copy constructor initializes an object using values of another object passed to it as a parameter.

Example:

class Cube
{
   int side;
   public:
   Cube()
   {
     side=10;
   }
   Cube(int x)
   {
      side=x;
   }
};

int main()
{
   Cube c1;
   Cube c2(20);
   cout ‹‹ c1.side;
   cout ‹‹ c2.side;
}
Output: 10  20


Destructor -
  • Destructor is a member function which destroys or deletes an object. It is used to free the resources occupied by an object.
  • It is called automatically when the object is no longer needed or when scope/lifetime of an object ends.
  • It has same name as that of class preceded by tilde(~) sign.
  • It is always declared or defined under public section.
  • It neither returns any value nor takes any argument.
  • It cannot be declared as static, const and virtual.
Syntax:
class A
{
 public:
 ~A();
};

Example:

class example
{
   public:
    ~example();  //destructor
};
