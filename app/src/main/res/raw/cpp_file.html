File - A file is a collection of bytes stored on a secondary storage device, which is generally a disk of some kind.
A file can be of 2 types, namely
1. Text file
2. Binary file
IMP Note:
1.Every file has a BOF (Beginning of the file) and EOF (End of the file) tags.
2.Header file #include‹fstream› must be added in the program to perform all file operations in C++.
Opening a file - Before a file can be used for reading or writing, it must be opened. This is can be done by using ifstream class for opening file in read mode or by using ofstream class for opening file in write mode or using fstream class which can used to open file in read as well as in write mode.
Syntax:
stream_obj.open("file_name",mode);

Using ifstream class:
ifstream f1;
f1.open("PLR.txt");

Using ofstream class:
ofstream f1;
f1.open("PLR.txt");

Using fstream class:
fstream f1;
f1.open("PLR.txt",ios::out | ios::binary);

Some file opening modes are:
  • ios::in - Open file for read only
  • ios::out - Open file for write only
  • ios::app - Append to end of file
  • ios::ate - Move to end of file
  • ios::binary - Open in binary mode
  • ios::trunc - Delete contents of existing file
  • ios::nocreate - Open fails if file does not exist
  • ios::noreplace - Open fails if file already exists
All these modes can be combined using bitwise OR(|).
Example:

#include‹iostream›
using namespace std;
#include‹fstream›
int main()
{
   fstream f1;
   f1.open("PLR.txt",ios::in | ios::binary);
   f1.close();
   return 0;
}
Here, PLR.txt is opened in read mode and binary mode. f1 is an object of class fstream.


Closing a file - A call to .close() closes a previously opened file.
Syntax:
file_obj.close();

Example:

f1.close();
*refer previous example.


File manipulation functions:
1)seekg() - Moves get pointer to specified location. For example,
ifstream f1;
f1.seekg(20);
Here we are giving the absolute position in file.
However we can move get pointer to a position relative to position specified in seekg().
These positions are specified as follows:
  • ios::beg - Start of file
  • ios::cur - Current position of pointer
  • ios::end - End of file
Example:

i]   f1.seekg(30,ios::beg); //go to byte no.30 from beginning of file
ii]  f1.seekg(-4,ios::cur);  //back up 4 bytes from current position
iii] f1.seekg(0,ios::end);   //go to end of file

2)seekp() - Moves put pointer to specified location.
Example:

ofstream f1,f2;
f1.seekp(20);     //moves put pointer to byte no.20
f2.seekp(30,ios::beg); //moves put pointer to byte no.30 from BOF

3)tellp() - Returns current position of the put pointer.
Example:

R=f1.tellp();  //returns and stores current position of put pointer

4)tellg() - Returns current position of the get pointer.
Example:

R=f1.tellg();  //returns and stores current position of get pointer
