1)For loop - For loop is best used when the number of iterations is known.

Syntax:
for(initialization ; condition ; increment/decrement)


Example:

int main()
{
  int a;
  for(a=0;a<5;a++)
  {
    cout<<a;
  }
  return 0;
}
Output: 01234

2)While loop - While loop is best used when the number of iterations is NOT known.

Syntax:
initialization;
while(condition)

Example:

int main()
{
  int a=0;
  while(a<5)
  {
    cout<<a;
    a++;
  }
  return 0;
}
Output: 01234

3)do-while loop - do-while loop is used when the statements inside the loop are to be executed atleast once.

Syntax:
initialization;
do
{
  statements;
}
while(condition);

Example:

int main()
{
  int a=7;
  do
  {
    cout<<a;
    a++;
  }
  while(a<5);
  return 0;
}
Output: 7
IMP Note: Here, even if the condition(a<5) is false still the statement is executed first and then the loop is terminated.
