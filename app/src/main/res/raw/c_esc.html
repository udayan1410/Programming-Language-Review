Escape Sequence Description
1. \n New line character
2. \a Audible signal
3. \v Vertical tab
4. \t Tab (Horizontal)
5. \' To represent single quotation
6. \" To represent double quotation
7. \\ To represent back slash
8. \? To represent a question mark
9. \r Carriage return
10. \b Backspace
11. \0 NULL
