Functions-
  • A function is a group of statements that together perform a task.
  • Every C program has at least one function, which is main(), and all the most trivial programs can define additional functions.
  • It is possible to divide up code into seperate functions and that code can be reused again and again.
  • A function can be also referred to as subroutine , procedure and method in other programming languages.
There are 3 important things in any function-
  • Function declaration
    • - It tells the compiler about a function's name, return type and parameters.
    Syntax- 
    return_type function_name(parameter)
{
   body of the function
}

  • Function Definition
    • - A function definition provides the actual body of the function.
    Syntax-
    void sum( int a ,int b ) {
   b = b+a;
   print(b)
}

  • Function Call
    • - Whenever a function is required the function can be accessed by calling the function and passing in the required parameters.
    Syntax-
    function_name(parameters_list);

Function Arguments-
  • Function argument is the data which we can pass to a function and perform calculation.The number of arguments can differ from function to function depending on the application.
    • Syntax-
    void function_name(par1,par2){
	return par1+par2;
}

Function Return types-
  • We can choose to return the data after computation from the function or we can display the result in the function itself.
    • Syntax-
    return-type function_name(){
function body
return return-type;
}
