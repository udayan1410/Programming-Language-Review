STL (Standard Template Library) -

The C++ STL is a set of general purpose template classes and functions.It provides us many widely used predefined algorithms and data structures like vectors, lists, stacks and queues.

The three main components of STL are:
1)Containers - Containers are objects that actually stores data. It manages the storage space for its elements and provides member functions to access them, either directly or through iterators.
Containers can be broadly classified into 3 types:
  • Sequence containers
  • Associative containers
  • Derived containers
2)Algorithms - STL algorithms are standard template functions, which support various operations. We can use these algorithms if we include <algorithm> in our program.
They are of following types:
  • Sorting algorithms
  • Set algorithms
  • Relational algorithms
  • Numeric algorithms
  • Mutating algorithms
  • Non-mutating algorithms
3)Iterators - Iterators are used to access container elements .
They are of following types :
  • Input
  • Output
  • Forward
  • Bidirectional
  • Random
