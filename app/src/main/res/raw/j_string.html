Java String:

In Java, string is basically an object that represents sequence of char values. An array of characters works same as Java string.
There are two ways to create String object:
  • By string literal

    • Syntax:
    //Using new keyword
	   char[] ch={'j','a','v','a','t','p','o','i','n','t'};  
	   String s=new String(ch);


  • By new keyword

    • Syntax:
    	 //Using string literal
	   String s="javatpoint";


Java String class methods to perform operations on string:
1. static String equalsIgnoreCase(String another): compares another string. It doesn't check case.
2. String concat(String str): concatenates the specified string.
3. boolean equals(Object another): checks the equality of string with the given object.
4. String[] split(String regex): returns a split string matching regex
5. int length(): returns string length
6. String replace(char old, char new): replaces all occurrences of the specified char value.
7. int indexOf(int ch, int fromIndex): returns the specified char value index starting with given index.
8. String intern(): returns an interned string.
9. String substring(int beginIndex): returns substring for given begin index.
10. String trim(): removes beginning and ending spaces of this string.
11. String toUpperCase(): returns a string in uppercase.
12. String toUpperCase(Locale l): returns a string in uppercase using specified locale.
Example:
					

public class StringExample
{  
  public static void main(String args[])
	{  
	  String s1="java";//creating string by java string literal  
	  char ch[]={'s','t','r','i','n','g','s'};  
	  String s2=new String(ch);//converting char array to string  
	  String s3=new String("example");//creating java string by new keyword  
	  System.out.println(s1);  
	  System.out.println(s2);  
	  System.out.println(s3);  
		}
	}
