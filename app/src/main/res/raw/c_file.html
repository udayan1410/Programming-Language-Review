File
A file is a collection of bytes stored on a secondary storage device, which is generally a disk of some kind.
A file can be of 2 types, namely
1. Text file
2. Binary file

IMP Note: Every file has a BOF (Beginning of the file) and EOF (End of the file) tags.

Opening a file -
Before a file can be used for reading or writing, it must be opened. This is done by a call to the fopen() function.
Syntax:
fopen("file_name","mode");

Some file opening modes are:
  • "r" (read) - open existing file for reading.
  • "w" (write) - open (create if necessary) file for writing, discard previous contents.
  • "a" (append) - open (create if need be), do not discard previous contents, write at the end of the file.
  • "rb" - open an existing binary file for reading.
Example:

int main()
{
   FILE *fp;
   fp=fopen("PLR.txt","w");
   fclose(fp);
   return 0;
}
Here, PLR.txt is opened in write mode. fp is a pointer variable of type FILE. "FILE" is an in-built structure datatype given in header file . All file manipulation operations in C require this datatype "FILE".

IMP Note: If fopen() is successful in opening the file, it returns a pointer of type FILE. If fopen() cannot successfully open the file, it returns a NULL pointer instead.

Closing a file -
A call to fclose() closes a previously opened file.
Syntax:
fclose(address of file);

Example:

fclose(fp);
*refer previous example.

Other important file related functions are:

1. fprintf() - Writes data to the file.
Syntax:
fprintf(file-pointer,"control string",list of arguments);

Example:

fprintf(f1,"%s%d",name,salary);
It writes name and salary in file.

2. fscanf() - Reads a set of data values from a file.
Syntax:
fscanf(file-pointer,"control string",address of arguments);

Example:

fscanf(f1,"%s%d",name,&salary);
It reads name and salary from the file.

3. getc() - Reads a character from a file.
Syntax:
getc(file-pointer);

Example:

char ch;
ch=getc(f1);
It reads a character from the file pointed by file-pointer f1.


4. getw() - Reads an integer from a file.
Syntax:
getw(file-pointer);

Example:

int num;
num=getw(f1);
It reads integer from the file pointed by file-pointer f1.

5. putw() - Writes an integer to a file.
Syntax:
putw(int number,file-pointer);

Example:

int c=2;
putw(c,f1);
Writes '2'to the file pointed by f1 pointer.

6. ftell() - It returns the current position of file pointer.
Syntax:
ftell(file pointer);

Example:

ftell(f1);
It returns the current position of file-pointer "f1".

7. fseek() - It is used to move the position of file pointer to the desired location within a file.
Syntax:
fseek(file-pointer,offset,position);
    Position modes:
  • 0/SEEK_SET - Beginning position of file(BOF)
  • 1/SEEK_CUR - The current position
  • 2/SEEK_END - End position of file (EOF)
Example:

fseek(f1,0,0); - Pointer will move to the starting position of the file.
fseek(f1,0,2); - Pointer will move to the end position of the file.
fseek(f1,5,0); - Pointer will move by '5+1' positions forward from the start of the file.
fseek(f1,3,1); - Pointer will move '3' positions forward from the current position.
fseek(f1,-3,1); - Pointer will move '3' positions backwards from the current position.
fseek(f1,-3,2); - Pointer will move '3+1' positions backwards from the end position.
fseek(f1,3,2); - Pointer will show an error as we cannot go beyond the end position of the file.
fseek(f1,-3,0); - Pointer will show an error as we cannot move backwards from the start position of the file.

IMP Note: Pointer will move 'n+1' positions for position modes: 0 and 2 (i.e SEEK_SET & SEEK_END) whereas it will move 'n' positions for position mode: 1 (i.e SEEK_CUR), where n is the offset by which pointer moves.
