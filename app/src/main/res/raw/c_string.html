String
-String is defined as an array of characters which is terminated with a special character called NULL (‘\0’).

Declaration of strings:
Declaring a string is as simple as declaring a one dimensional array. Below is the basic syntax for declaring a string.
Syntax:
char str_name[size];

In the above syntax, "str_name" is any name given to the string variable and "size" is the length of the string, of which '\0' will take one space, therefore the total size of the string declared should always be "n+1" where n is the no. of the characters of string and plus one for '\0' character.

Initializing a String:
Syntax:
1. char str[] = "PLR"; 
2. char str[50] = "PLR"; 
3. char str[] = {'P','L','R','\0'};
4. char str[14] = {'P','L','R','\0'};

Example:

int main()
{
    char a[20];
    printf("Enter string: ");
    scanf("%s",a);
    printf("Your string is: %s",a);
    return 0;
}
Output: Enter string: Programming
        Your string is: Programming

Passing strings to function:
As strings are character arrays, we can pass strings to function in the same way we pass an array to a function.

Example:

void display(char str[]) 
{ 
    printf("String is : %s",str); 
} 
int main() 
{    
    // declare and initialize string 
    char str[] = "PLR"; 
    // print string by passing string 
    // to a different function 
    display(str); 
    return 0; 
} 
Output:
String is : PLR
