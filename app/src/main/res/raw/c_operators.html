1] Arithmetic Operators

  • '+' - The '+' operator adds two operands. For example, a+b.
  • '-' - The '-' operator subtracts two operands. For example, a-b.
  • '*' - The '*' operator multiplies two operands. For example, a*b.
  • '/' - The '/' operator divides the first operand by the second. For example, a/b.
  • '%' - The '%' operator returns the remainder when first operand is divided by the second. For example, a%b.
  • i++ - Post-increment operator
  • ++i - Pre-increment operator
  • i-- - Post-decrement operator
  • --i - Pre-decrement operator

2] Relational Operators

  • '==' - Equal to (Example: 5==3 returns 0)
  • '!=' - Not equal to (Example: 5!=3 returns 1)
  • '>' - Greater than (Example: 5>3 returns 1)
  • '<' - Less than (Example: 5<3 returns 0)
  • '>=' - Greater than or equal to (Example: 5>=3 returns 1)
  • '<=' - Less than or equal to (Example: 5<=3 returns 0)
IMP Note: If the result is true, the relational operators return 1 and if it is false then the operators return 0.

3] Logical Operators

  • && - Logical AND operator.True(1) only if all operands are true.
  • || - Logical OR operator.True(1) only if either one operand is true.
  • ! - Logical NOT operator.True(1) only if the operand is 0.

4] Bitwise Operators

  • & - Bitwise AND
  • | - Bitwise OR
  • ^ - Bitwise Exclusive OR
  • ~ - Bitwise complement
  • >> - Shift right
  • << - Shift left

5] Assignment Operators

  • '=' - Assignment operator
  • '+=' - a+=b, it is same as a=a+b
  • '-=' - a-=b, it is same as a=a-b
  • '*=' - a*=b, it is same as a=a*b
  • '/=' - a/=b, it is same as a=a/b
